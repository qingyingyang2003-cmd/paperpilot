# Rapid scanning method for SICM based on autoencoder network
- 作者：Wenlin Wu et al.
- 期刊/会议：Micron
- 年份：2024
- DOI：10.1016/j.micron.2023.103579
## 一段话总结
本文提出了一种基于自编码器（autoencoder）网络的SICM（扫描离子电导显微镜）快速扫描方法，旨在解决SICM跳跃扫描模式（hopping mode）中时间分辨率低的问题。核心思路是：先以低分辨率（欠采样）快速扫描获取图像，再通过自构建的primitive-autoencoder超分辨率网络将低分辨率图像重建为高分辨率图像，最后利用重建图像推断出优化的扫描路径，从而在实际高分辨率扫描时跳过冗余的探针回升步骤。实验结果表明，与传统插值方法相比，该方法的平均PSNR（峰值信噪比）提升超过7.5823 dB，平均SSIM（结构相似性指数）提升超过0.2372，扫描速度提升156.25%。

## 摘要翻译
SICM（扫描离子电导显微镜）能够对活细胞进行无损成像，在生命科学、医学、药理学等众多领域具有重要价值。然而，由于SICM跳跃扫描模式中回程高度的不确定性，其时间分辨率相对较低，难以满足动态扫描的需求。针对上述问题，本文提出了一种基于自编码器网络的SICM快速扫描方法。首先，将欠采样图像切分为小图像块序列；其次，将其输入自构建的primitive-autoencoder超分辨率网络以计算高分辨率图像；最后，利用计算所得图像推断扫描路径，重建真实的高分辨率扫描路径。实验结果表明，所提网络能够在多种低分辨率扫描图像的超分辨率任务中重建出更高分辨率的图像。与现有传统插值方法相比，平均峰值信噪比提升超过7.5823 dB，平均结构相似性指数提升超过0.2372。同时，使用本文方法进行高分辨率图像扫描，与传统方法相比扫描速度提升156.25%。这为实现SICM对动态样品的高时间分辨率成像开辟了可能，并将进一步推动SICM在未来的广泛应用。

## 框架
本文的研究框架遵循"问题识别 → 方法设计 → 实验验证"的逻辑主线，具体如下：

**1. 问题识别**
- SICM跳跃扫描模式中，探针每次测量后需返回安全高度，导致大量时间浪费在冗余的垂直运动上
- 现有改进方法（改进扫描模式、压缩感知等）鲁棒性较差，难以同时兼顾速度与精度

**2. 核心思路**
- 将"提高扫描速度"问题转化为"图像超分辨率重建 + 路径规划"问题
- 先用低分辨率（大步长）快速粗扫，再用深度学习网络补全高分辨率细节，最后用重建结果指导精扫路径

**3. 方法设计（三个模块）**
- **扫描高度-图像像素映射**：将SICM高度数据归一化映射为图像像素值，为后续深度学习提供数据基础
- **Primitive-Autoencoder超分辨率网络**：
  - Block-based primitive filter（基于块的原始滤波器）：将图像切分为4×4小块
  - Autoencoder网络：编码器提取特征，解码器重建，全连接层完成2倍超分辨率
  - Skip connection（跳跃连接）：保留细节特征，提升鲁棒性
- **非线性滤波后处理**：中值滤波 + 双边滤波，消除块间边缘噪声，融合全局特征

**4. 实验验证**
- 消融实验：验证skip connection和滤波模块的必要性
- 超分辨率对比实验：与区域线性、双线性、三次样条插值方法对比
- 扫描速度对比实验：验证路径规划后的实际加速效果

## 研究问题
**核心问题：** 如何在不牺牲图像质量的前提下，大幅提升SICM的时间分辨率（扫描速度）？

**问题的重要性：**
SICM是目前少数能够对活细胞进行非接触、非破坏性纳米级成像的工具，在研究细胞动态行为（如膜形态变化、药物响应等）方面具有不可替代的价值。然而，其跳跃扫描模式的固有缺陷——探针每次测量后必须回升至安全高度——导致大量时间浪费在无效的垂直运动上，使得SICM无法捕捉快速变化的细胞动态过程。

**现有方法的不足（知识空白）：**
- 改进扫描控制策略的方法（如Zhuang等人的连续控制模式、Jung等人的闭环ARS模式）依赖样品特性，鲁棒性差
- 压缩感知方法（Li等人）在复杂样品上泛化能力有限
- 基于预测的方法（Takahashi等人）仅适用于特定目标区域
- 上述方法均从硬件控制层面入手，未充分利用图像后处理与深度学习的潜力

**本文填补的空白：**
将深度学习超分辨率技术与SICM扫描路径规划相结合，从"软件+算法"层面提升时间分辨率，提供了一种与硬件改进互补的新思路。

## 方法
**整体流程：**

**第一步：数据预处理（高度-像素映射）**
- 将SICM采集的高度数据通过公式 P(x,y) = 255 × h(x,y)/H 映射为图像像素值
- 使用max-min归一化消除映射特征损失，确保图像完整表达扫描高度信息

**第二步：欠采样低分辨率图像获取**
- 以步长为2的跳跃扫描模式快速获取低分辨率图像（如16×16或64×64）
- 对应的高分辨率参考图像以步长为1扫描获得（如32×32或128×128）

**第三步：Primitive-Autoencoder网络超分辨率重建**

*Block-based primitive filter（基于块的原始滤波器）：*
- 使用n×n滑动窗口提取图像块，训练时步长s=1，预测时步长s=n
- 将图像分解为4×4的小块序列输入网络

*Autoencoder网络结构：*
- 编码器：3×3和2×2卷积层 + BN（批归一化）层 + Leaky-ReLU激活
- 解码器：2×2和3×3转置卷积 + BN层 + Leaky-ReLU激活
- 编解码器之间：1×1卷积作为非线性跳跃连接层
- 全连接层：三层全连接 + Leaky-ReLU，输出尺寸为输入的r=2倍（实现2倍超分辨率）
- 单层skip connection：将输入图像块经Leaky-ReLU后与解码器特征图融合，增强细节表达

*损失函数：*
- 使用MSE（均方误差）作为损失函数，对比重建的8×8图像块与真实8×8图像块

**第四步：非线性滤波后处理**
- 中值滤波：消除块边缘计算噪声，保留边缘特征
- 双边滤波：基于高斯分布加权平均，同时考虑空间距离和像素值差异，进一步保边去噪

**第五步：扫描路径规划与高分辨率扫描**
- 利用超分辨率重建图像推断每个扫描点的预测高度 Hp_s
- 设置安全高度阈值 ΔH，探针直接跳至计算高度而非固定安全高度
- 实际高分辨率扫描时跳过冗余的回升路径，大幅缩短扫描时间

**实验设计：**
- 消融实验：对比有/无skip connection、有/无非线性滤波的四种组合，重复5次取平均
- 超分辨率对比实验：在PANC-1（人胰腺癌细胞）、人关节软骨表面细胞、大鼠嗜碱性白血病细胞及静态物体上测试，与区域线性插值、双线性插值、三次样条插值对比
- 扫描速度对比实验：在大鼠嗜碱性白血病细胞上进行5次重复扫描，对比传统方法与本文方法的平均扫描时间
- 评价指标：PSNR（峰值信噪比）和SSIM（结构相似性指数）

### 关键实验参数
- 图像块输入尺寸：4×4 像素
- 超分辨率倍数 (r)：2倍（输出8×8像素）
- 训练时滑动窗口步长 (s)：1
- 预测时滑动窗口步长 (s)：n（等于窗口尺寸）
- 填充 (padding)：0
- 编码器卷积核尺寸：3×3 和 2×2
- 解码器转置卷积核尺寸：2×2 和 3×3
- 跳跃连接卷积核尺寸：1×1
- 全连接层数量：3层
- 激活函数：Leaky-ReLU
- 归一化方法：Batch Normalization (BN)
- 损失函数：MSE（均方误差）
- 训练数据下采样方法：三次样条插值（cubic spline interpolation）
- 低分辨率图像尺寸（消融实验）：16×16（步长2扫描）
- 高分辨率参考图像尺寸（消融实验）：32×32（步长1扫描）
- 低分辨率图像尺寸（全图超分辨率）：64×64
- 高分辨率目标图像尺寸（全图超分辨率）：128×128
- 局部超分辨率输入尺寸：32×32
- 局部超分辨率输出尺寸：64×64
- 实验重复次数：5次
- 扫描分辨率（速度对比实验）：32×48
- 传统方法扫描时间：1935.0000 秒
- 本文方法扫描时间：846.5625 秒

## 关键结果
**1. 消融实验结果**
- 仅去除skip connection层：平均PSNR = 17.5044 dB，平均SSIM = 0.4567，图像模糊
- 去除滤波（保留skip connection）：平均PSNR = 19.8489 dB，平均SSIM = 0.5400，轮廓更清晰但噪声明显
- 完整方法（skip connection + 滤波）：平均PSNR = 27.8422 dB，平均SSIM = 0.8178
- 完整方法相比仅去除滤波：PSNR提升7.9933 dB，SSIM提升0.2787
- 结论：skip connection和非线性滤波均为不可缺少的关键组件

**2. 超分辨率对比实验（PANC-1全图，Fig.6a）**
- 区域线性插值：PSNR = 29.58 dB，SSIM = 0.90
- 双线性插值：PSNR = 29.12 dB，SSIM = 0.87
- 三次样条插值：PSNR = 30.11 dB，SSIM = 0.90
- 本文方法：PSNR = 31.71 dB，SSIM = 0.94
- 本文方法相比最优插值方法（三次样条）：PSNR提升1.60 dB，SSIM提升0.04

**3. 超分辨率对比实验（PANC-1全图平均，Fig.6b）**
- 区域线性插值：平均PSNR = 21.70 dB，平均SSIM = 0.88
- 双线性插值：平均PSNR = 22.28 dB，平均SSIM = 0.87
- 三次样条插值：平均PSNR = 23.24 dB，平均SSIM = 0.90
- 本文方法：平均PSNR = 23.72 dB，平均SSIM = 0.92
- 本文方法相比最优插值方法：PSNR提升0.48 dB，SSIM提升0.02

**4. 超分辨率对比实验（PANC-1局部图像，Fig.6c）**
- 三次样条插值（最优插值）：平均PSNR = 24.85 dB，平均SSIM = 0.88
- 本文方法：平均PSNR = 25.90 dB，平均SSIM = 0.91
- 本文方法相比最优插值方法：PSNR提升1.05 dB，SSIM提升0.03

**5. 与传统方法总体对比**
- 相比所有传统插值方法，平均PSNR提升 > 7.5823 dB，平均SSIM提升 > 0.2372

**6. 扫描速度对比实验（大鼠嗜碱性白血病细胞，32×48分辨率）**
- 传统方法扫描时间：1935.00 秒
- 本文方法扫描时间：846.56 秒
- 速度提升：156.25%，扫描时间缩短约56.3%

## 创新点
**1. 将超分辨率重建与扫描路径规划相结合的新范式**
现有SICM加速方法主要从硬件控制和扫描模式改进入手，本文首次将深度学习图像超分辨率技术引入SICM扫描流程，形成"粗扫→AI重建→精扫路径规划"的新范式，从软件算法层面解决时间分辨率问题，与硬件改进方向互补。

**2. 针对SICM图像特性设计的Primitive-Autoencoder网络**
网络设计充分考虑了SICM图像的"灰度锐化"和显著边缘梯度特性，采用基于图像块（patch-based）的处理方式，而非对整幅图像进行超分辨率计算。这种设计在保留局部细节特征的同时降低了计算复杂度，并增强了网络对不同样品的泛化能力。

**3. 单层Skip Connection的特征融合设计**
在标准autoencoder结构基础上，额外设计了一个将输入图像块直接与解码器输出融合的单层跳跃连接，公式为 g(x) = α + LeakyReLU(x)。这一设计专门针对SICM图像中的微小变化和局部细节特征，有效提升了网络对细节的敏感性和鲁棒性。

**4. 中值滤波与双边滤波的组合后处理策略**
针对基于图像块的超分辨率方法在块间边缘处产生计算噪声的固有问题，提出了中值滤波（消除块边缘噪声）与双边滤波（保边去噪）的组合策略，有效解决了块间特征融合问题，消融实验证明该策略带来了7.99 dB的PSNR提升。

**5. 基于计算高度的动态安全高度阈值机制**
传统跳跃扫描模式使用固定安全高度，本文利用超分辨率重建图像预测每个扫描点的高度，并设置动态安全阈值 ΔH，使探针直接跳至预测高度附近，从根本上消除了冗余的垂直运动，实现了156.25%的扫描速度提升。

## 局限性
**1. 超分辨率倍数有限**
本文仅实现了2倍超分辨率（4×4→8×8图像块），对于需要更高分辨率提升的应用场景（如4倍、8倍），方法的有效性尚未验证，且更高倍数的超分辨率重建误差可能对路径规划精度产生更大影响。

**2. 安全高度阈值 ΔH 的设定依赖经验**
论文中提到设置安全高度阈值以防止计算误差损坏探针，但未详细说明 ΔH 的具体取值方法和自适应策略。对于表面形貌复杂或变化剧烈的样品，固定阈值可能不够安全或过于保守。

**3. 网络训练数据依赖性**
模型使用SICM实际扫描图像训练，其泛化能力依赖于训练数据的多样性。对于训练集中未见过的细胞类型或形貌特征，网络的重建精度可能下降，论文虽在多种细胞上进行了测试，但样本种类仍然有限。

**4. 图像尺寸较小**
实验中使用的图像分辨率较低（最大128×128），在实际应用中，SICM往往需要扫描更大区域或更高分辨率的图像，方法在大尺寸图像上的性能和计算效率未经充分验证。

**5. 动态细胞的时序一致性问题**
对于快速变化的动态细胞，粗扫阶段与精扫阶段之间存在时间间隔，细胞形态可能已发生变化，导致超分辨率重建的预测高度与实际高度不符。论文未对这一时序误差进行定量分析。

**6. 与其他深度学习超分辨率方法的对比缺失**
论文仅与传统插值方法（区域线性、双线性、三次样条）进行了对比，未与其他深度学习超分辨率方法（如SRCNN、ESRGAN等）进行比较，难以全面评估所提网络的竞争力。

**7. 计算时间未充分讨论**
论文重点展示了扫描速度的提升，但未详细报告超分辨率网络推断所需的计算时间，若推断时间较长，实际的总体时间收益可能低于156.25%。
