# Example Outputs

This directory contains sample outputs generated by PaperPilot, demonstrating what the tool produces for real scientific papers. All notes below were generated from SICM (Scanning Ion Conductance Microscopy) and precision measurement research papers.

## Files

| File | Paper | Journal | Year |
|------|-------|---------|------|
| `nanoscale-surface-charge-hair.md` | Nanoscale Surface Charge Visualization of Human Hair | Analytical Chemistry | 2019 |
| `highspeed-sicm-for-the.md` | High-Speed SICM for Visualization of Nanoscale Dynamic Structural Changes in Hippocampal Neurons | Analytical Chemistry | 2020 |
| `rapid-scanning-method-for.md` | Rapid Scanning Method for SICM Based on Autoencoder Network | Micron | 2024 |
| `study-on-contactfree-scanning.md` | Contact-Free Scanning and Imaging Reliability for Low Electric Field SICM With Dual-Barrel Pipette | IEEE Sensors Journal | 2024 |
| `realtime-observation-of-tunneling.md` | Real-Time Observation of Tunneling Nanotube Dynamics Using SICM and SECM | Analytical Chemistry | 2025 |
| `cross-scale-positioning-planar.md` | Cross-Scale Positioning of Three DOF Planar Motions With Subnanometer Resolution | IEEE Trans. Instrum. Meas. | 2025 |

Each note contains 8 structured sections generated by the Analyst Agent:
- One-paragraph summary (一段话总结)
- Abstract translation (摘要翻译)
- Research framework (框架)
- Research question (研究问题)
- Methods with key parameters (方法 + 关键实验参数)
- Key results (关键结果)
- Innovations (创新点)
- Limitations (局限性)
- References to follow (值得追踪的参考文献)

## How to generate your own

```bash
# Set your API key
export ANTHROPIC_API_KEY=sk-ant-xxx

# Analyze any PDF — place your own PDF in this directory or use an absolute path
paperpilot read your-paper.pdf -o ./notes

# Search and fetch papers
paperpilot fetch "your research topic" --download --analyze
```

> Note: PDF files and extracted figures are excluded from version control to keep the repo lightweight. Run `paperpilot read` on your own papers to generate outputs locally.
