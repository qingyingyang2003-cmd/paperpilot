# Rapid scanning method for SICM based on autoencoder network
- 作者：Wenlin Wu et al.
- 期刊/会议：Micron
- 年份：2024
- DOI：10.1016/j.micron.2023.103579
## 一段话总结
本文提出了一种基于自编码器（autoencoder）网络的SICM（扫描离子电导显微镜）快速扫描方法，旨在解决SICM跳跃扫描模式（hopping mode）中时间分辨率低的问题。核心思路是：先以低分辨率（欠采样）快速扫描获取图像，再通过自构建的primitive-autoencoder超分辨率网络将低分辨率图像重建为高分辨率图像，最后利用重建图像推断出优化的扫描路径，从而在实际高分辨率扫描时跳过冗余的探针回退路径。实验结果表明，与传统插值方法相比，该方法的平均PSNR（峰值信噪比）提升超过7.5823 dB，平均SSIM（结构相似性指数）提升超过0.2372，扫描速度提升156.25%。

## 摘要翻译
SICM（扫描离子电导显微镜）能够对活细胞进行无损成像，在生命科学、医学、药理学等众多领域具有重要价值。然而，由于SICM跳跃扫描模式中探针回退高度的不确定性，其时间分辨率相对较低，难以满足动态扫描的需求。针对上述问题，本文提出了一种基于自编码器网络的SICM快速扫描方法。首先，将欠采样图像切分为小图像块序列；其次，将其输入自构建的primitive-autoencoder超分辨率网络以计算高分辨率图像；最后，利用计算所得图像推断扫描路径，重建真实的高分辨率扫描路径。实验结果表明，所提网络能够在多种低分辨率扫描图像的超分辨率任务中重建出更高分辨率的图像。与现有传统插值方法相比，平均峰值信噪比提升超过7.5823 dB，平均结构相似性指数提升超过0.2372。同时，使用本文方法进行高分辨率图像扫描，与传统方法相比扫描速度提升156.25%。该方法为SICM实现动态样品的高时间分辨率成像开辟了可能，并将进一步推动SICM的广泛应用。

## 框架
本文的研究框架遵循"问题识别 → 方法设计 → 实验验证"的逻辑主线，具体如下：

**1. 问题识别**
- SICM跳跃扫描模式中，探针每次测量后需返回安全高度，导致大量时间浪费在冗余的垂直运动上
- 现有改进方法（改进扫描模式、压缩感知等）鲁棒性较差，难以同时兼顾速度与精度

**2. 核心思路**
- 将"提高扫描速度"问题转化为"图像超分辨率重建 + 路径规划"问题
- 先用低分辨率（大步长）快速粗扫，再用深度学习网络补全高分辨率细节，最后用重建结果指导精扫路径

**3. 方法设计（三个模块）**
- **扫描高度-图像像素映射**：将SICM高度数据归一化映射为图像像素值，为后续深度学习提供数据基础
- **Primitive-Autoencoder超分辨率网络**：
  - Block-based primitive filter（基于块的原始滤波器）：将图像切分为4×4小块
  - Autoencoder网络：编码器提取特征，解码器重建，全连接层完成2倍超分辨率
  - Skip connection（跳跃连接）：保留细节特征，提升鲁棒性
  - 非线性滤波组合（中值滤波 + 双边滤波）：消除块间边缘噪声，融合全局特征
- **路径规划与高分辨率扫描**：利用超分辨率图像推断各点计算高度，替代固定安全高度，减少探针垂直行程

**4. 实验验证**
- 消融实验（ablation experiment）：验证skip connection和滤波模块的必要性
- 超分辨率对比实验：与区域线性插值、双线性插值、三次样条插值对比
- 扫描速度对比实验：量化时间提升效果

## 研究问题
**核心问题：** 如何在不牺牲图像质量的前提下，大幅提升SICM的时间分辨率（扫描速度）？

**问题的重要性：**
SICM能够对活细胞进行非接触、非破坏性的纳米级形貌成像，在研究细胞动态过程（如膜结构变化、药物响应等）方面具有独特优势。然而，跳跃扫描模式要求探针在每个测量点后都回退到固定安全高度，这一冗余运动消耗了大量时间，导致SICM无法捕捉快速变化的细胞动态行为，严重制约了其在动态生物学研究中的应用。

**现有方法的不足（知识空白）：**
- 改进扫描模式的方法（如Zhuang等人引入水平运动、Jung等人的闭环ARS模式）依赖对样品形貌的实时预测，对复杂样品鲁棒性差
- 压缩感知方法（Li等人）在稀疏重建上有局限，泛化能力有限
- 上述方法均在扫描过程中进行干预，难以同时保证速度和精度

**本文填补的空白：**
将后处理图像超分辨率技术与SICM扫描路径规划相结合，提出了一种与扫描模式解耦的加速框架，使得速度提升不依赖于对特定样品形貌的假设，从而具有更好的泛化性。

## 方法
**整体流程：**

**第一步：数据预处理（高度-像素映射）**
- 将SICM采集的高度数据h(x,y)通过公式P(x,y) = 255 × h(x,y)/H映射为灰度图像像素值
- 使用max-min归一化消除映射特征损失，确保图像特征与扫描高度保持相对关系

**第二步：欠采样图像获取**
- 以2倍扫描步长（scanning step = 2）进行快速粗扫，获得低分辨率图像（如16×16或64×64）
- 正常扫描步长为1，对应高分辨率图像（如32×32或128×128）

**第三步：Primitive-Autoencoder网络超分辨率重建**

*Block-based primitive filter（基于块的原始滤波器）：*
- 使用n×n滑动窗口提取图像块，训练时步长s=1，预测时步长s=n
- 将图像分解为4×4的小块序列输入网络

*Autoencoder网络结构：*
- 输入尺寸：4×4图像块
- 编码器：3×3卷积层 + BN（批归一化）+ Leaky-ReLU，接2×2卷积层
- 解码器：2×2转置卷积 + BN + Leaky-ReLU，接3×3转置卷积
- 编解码器之间：1×1卷积层作为非线性skip connection，融合解码特征
- 全连接层：三层全连接 + Leaky-ReLU，输出尺寸为输入的r=2倍（即8×8）
- 单层skip connection：将输入图像块经Leaky-ReLU后与解码器特征图融合，保留细节

*损失函数：*
- 使用MSE（均方误差）作为损失函数，对比8×8预测块与真实8×8扫描块

*训练数据：*
- 使用SICM实际扫描的真实图像，将8×8块下采样至4×4作为输入，8×8块作为ground truth

**第四步：非线性滤波后处理**
- 中值滤波（median filter）：平滑块间边缘噪声，保留边缘特征
- 双边滤波（bilateral filter）：基于高斯分布加权平均，同时考虑空间距离和像素值差异，进一步保留边缘并融合特征

**第五步：路径规划与高分辨率扫描**
- 利用超分辨率图像推断各扫描点的计算高度Hp_s
- 设置安全高度阈值ΔH防止计算误差损坏探针
- 探针直接跳至计算高度而非固定安全高度，减少垂直行程，实现加速

**实验验证设计：**
- 消融实验：对比有/无skip connection、有/无非线性滤波的四种组合，各重复5次
- 超分辨率对比实验：与区域线性插值（Inner Area）、双线性插值（Inner Linear）、三次样条插值（Inner Cubic）对比，测试对象包括人胰腺癌细胞（PANC-1）、人关节软骨表面细胞、大鼠嗜碱性白血病细胞及静态物体
- 扫描速度对比实验：以大鼠嗜碱性白血病细胞为对象，32×48分辨率，重复5次取平均
- 评价指标：PSNR（峰值信噪比）和SSIM（结构相似性指数）

### 关键实验参数
- 图像块输入尺寸：4×4 像素
- 超分辨率倍数 (r)：2倍（输出8×8）
- 编码器卷积核尺寸：3×3, 2×2
- 解码器转置卷积核尺寸：2×2, 3×3
- 瓶颈层卷积核尺寸：1×1
- 激活函数：Leaky-ReLU
- 归一化方式：Batch Normalization (BN)
- 损失函数：MSE（均方误差）
- 训练时滑动窗口步长 (s)：1
- 预测时滑动窗口步长 (s)：n（等于窗口尺寸）
- Padding：0
- 低分辨率扫描步长：2（欠采样）
- 高分辨率扫描步长：1（正常采样）
- 消融实验图像尺寸（低分辨率）：16×16
- 消融实验图像尺寸（高分辨率参考）：32×32
- 全图超分辨率实验输入尺寸：64×64
- 全图超分辨率实验输出尺寸：128×128
- 局部超分辨率实验输入尺寸：32×32
- 局部超分辨率实验输出尺寸：64×64
- 扫描速度对比实验分辨率：32×48
- 传统方法扫描时间：1935.0000 s
- 本文方法扫描时间：846.5625 s
- 实验重复次数：5次

## 关键结果
**1. 消融实验结果**
- 仅去除skip connection层：平均PSNR = 17.5044 dB，平均SSIM = 0.4567（图像模糊）
- 去除滤波但保留skip connection：平均PSNR = 19.8489 dB，平均SSIM = 0.5400（轮廓较清晰但噪声多）
- 完整方法（含skip connection + 滤波）：平均PSNR = 27.8422 dB，平均SSIM = 0.8178
- 完整方法相比无滤波版本：PSNR提升7.9933 dB，SSIM提升0.2787，证明两个模块均不可或缺

**2. PANC-1细胞全图超分辨率实验（64×64 → 128×128）**
- 区域线性插值：PSNR = 29.5800 dB，SSIM = 0.9000
- 双线性插值：PSNR = 29.1200 dB，SSIM = 0.8700
- 三次样条插值：PSNR = 30.1100 dB，SSIM = 0.9000
- 本文方法：PSNR = 31.7100 dB，SSIM = 0.9400
- 相比最优插值方法（三次样条），PSNR提升1.6 dB，SSIM提升0.04

**3. PANC-1细胞全图超分辨率对比实验（平均值，Fig.6b）**
- 区域线性插值：平均PSNR = 21.7000 dB，平均SSIM = 0.8800
- 双线性插值：平均PSNR = 22.2800 dB，平均SSIM = 0.8700
- 三次样条插值：平均PSNR = 23.2400 dB，平均SSIM = 0.9000
- 本文方法：平均PSNR = 23.7200 dB，平均SSIM = 0.9200

**4. PANC-1细胞局部超分辨率实验（32×32 → 64×64）**
- 三次样条插值（最优插值）：平均PSNR = 24.8500 dB，平均SSIM = 0.8800
- 本文方法：平均PSNR = 25.9000 dB，平均SSIM = 0.9100
- 相比最优插值方法，PSNR提升1.05 dB，SSIM提升0.03

**5. 综合超分辨率性能（与所有传统插值方法相比）**
- 平均PSNR提升 > 7.5823 dB
- 平均SSIM提升 > 0.2372

**6. 扫描速度对比实验（大鼠嗜碱性白血病细胞，32×48分辨率）**
- 传统方法扫描时间：1935.0000 s
- 本文方法扫描时间：846.5625 s
- 速度提升：156.25%，约节省超过一半的扫描时间

## 创新点
**1. 将超分辨率重建与SICM扫描路径规划相结合的新框架**
这是本文最核心的创新点。以往的SICM加速方法都聚焦于改进扫描模式本身（如预测形貌、闭环控制），而本文将问题解耦：先用深度学习从欠采样图像重建高分辨率图像，再用重建结果指导实际扫描路径，从根本上减少了探针的冗余垂直运动。这种"先算后扫"的思路与现有方法有本质区别。

**2. 针对SICM图像特性设计的Primitive-Autoencoder网络**
作者注意到SICM图像具有"灰度锐化"和显著边缘梯度变化的特点，专门设计了基于图像块（patch-based）的自编码器网络。相比直接对整幅图像做超分辨率，以4×4小块为单位处理能更好地捕捉局部细节特征，同时降低计算复杂度。

**3. 单层Skip Connection的设计**
在标准autoencoder基础上，额外设计了一个将输入图像块（经Leaky-ReLU）与解码器特征图直接融合的skip connection层，专门用于增强网络对SICM图像中细微变化和局部精细特征的敏感性，提升了泛化能力和鲁棒性。

**4. 中值滤波与双边滤波的组合后处理策略**
针对基于图像块的超分辨率方法在块间边界处容易产生拼接噪声的问题，提出了中值滤波（去噪平滑）+ 双边滤波（保边融合）的两步非线性滤波组合策略，有效解决了块间特征不连续的问题，这在SICM超分辨率场景中是新颖的应用。

**5. 量化的扫描时间模型**
文中给出了传统跳跃扫描模式和改进方法的扫描时间数学模型（公式10-12），为理解速度提升的来源提供了理论依据，也为后续研究提供了可参考的分析框架。

## 局限性
**1. 超分辨率倍数固定为2倍**
当前方法仅实现了2倍超分辨率（4×4 → 8×8），对于需要更高倍率加速（如4倍、8倍）的应用场景，方法的有效性尚未验证。更高倍率的欠采样可能导致重建质量急剧下降。

**2. 网络泛化性依赖训练数据分布**
模型使用SICM实际扫描图像训练，对于形貌差异极大的新型样品（如具有极深沟槽或极高纵横比结构的样品），网络的泛化能力可能受限，需要针对性地补充训练数据。

**3. 安全高度阈值ΔH的设定缺乏自适应机制**
文中提到需要设置安全高度阈值ΔH以防止计算误差损坏探针，但未详细说明如何根据不同样品或超分辨率重建误差自适应地确定该阈值，这在实际应用中存在一定风险。

**4. 图像尺寸较小，实际应用场景有限**
实验中使用的图像分辨率较低（最大128×128），对于需要大视野高分辨率成像的实际生物学研究场景，方法的可扩展性和计算效率尚需验证。

**5. 与其他深度学习超分辨率方法缺乏对比**
文中仅与传统插值方法（区域线性、双线性、三次样条）进行了比较，未与其他深度学习超分辨率方法（如SRCNN、ESRGAN等）进行对比，难以全面评估所提网络的竞争力。

**6. 动态细胞成像的实时性未充分验证**
论文虽声称方法有助于动态样品的高时间分辨率成像，但实验主要在静态或准静态条件下进行，对于真正快速变化的动态生物过程（如细胞膜的实时形变），方法的实际效果尚未得到充分验证。

**7. 计算时间未纳入总体时间评估**
速度提升156.25%的结论仅考虑了扫描时间，未将超分辨率网络推断所需的计算时间纳入总体时间对比，可能高估了实际的时间效益。
